CIV7 Tracker — Favicon & Open Graph Pack

Files:
- img/favicon-256.png (PNG favicon you can reference directly)
- img/favicon-master-1024.png (high-res master source; optional)
- img/og-card.png (Open Graph image for social previews)
- site.webmanifest (optional; keep at repo root)
- docs/head-snippet.html (paste inside <head> on each page)

Usage:
1) Place your favicon bundle (favicon.ico, 16x16, 32x32, apple-touch-icon, site.webmanifest) at the repo ROOT.
   - If you don’t have site.webmanifest, use the one in this pack.
2) Copy img/og-card.png to /img/ (create folder if needed).
3) On each HTML page, paste docs/head-snippet.html into the <head> section.
4) Commit and push. Hard refresh the live site (Ctrl+F5) to verify favicon.
5) Test Open Graph by pasting your site URL in a chat app (Discord/Twitter).
